Este repositorio reúne una serie de prácticas y ejercicios diseñados para dominar las bases del lenguaje Python. El contenido abarca desde la sintaxis inicial hasta la implementación de lógica compleja mediante estructuras de control y funciones.

## 📁 Contenido del Repositorio

### 1. Lenguaje de Programación Python (Bases)
**Archivo:** `Hands_On_Lenguaje_de_Programación_Python.ipynb`
Introducción a los componentes esenciales del lenguaje:
* **Variables y Tipos de Datos:** Manejo de strings, enteros, flotantes y booleanos.
* **Estructuras de Datos:** Uso de listas y diccionarios para la creación de una agenda de contactos básica.
* **Operadores:** Implementación de operadores aritméticos, comparativos y lógicos.

### 2. Estructuras de Control de Flujo
**Archivo:** `Hands_On_Estructuras_de_Control_de_Flujo.ipynb`
Desarrollo de la lógica de decisión y repetición en el código:
* **Selección (if/elif/else):** Creación de flujos condicionales para la toma de decisiones.
* **Iteración (for/while):** Automatización de tareas repetitivas y recorrido de estructuras de datos.
* **Lógica Aplicada:** Resolución de problemas mediante la combinación de condiciones y bucles.

### 3. Introducción a las Funciones
**Archivo:** `Hands_On_Introducción_a_las_Funciones.ipynb`
Modularización y optimización del código:
* **Definición de Funciones:** Creación de bloques de código reutilizables con `def`.
* **Argumentos y Retornos:** Manejo de parámetros de entrada y salida de información.
* **Proyecto Integrador:** Desarrollo de un menú interactivo para una agenda, utilizando funciones para cada operación del sistema.
